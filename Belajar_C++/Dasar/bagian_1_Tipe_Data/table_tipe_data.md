| **Tipe Data**      | **Ukuran Memori** | **Nilai Minimum**                      | **Nilai Maksimum**                      | **Keterangan**                                                                                 |
|---------------------|-------------------|----------------------------------------|-----------------------------------------|---------------------------------------------------------------------------------------------|
| **`bool`**         | 1 byte           | `false` (0)                           | `true` (1)                             | Tipe logika untuk menyimpan nilai benar/salah.                                              |
| **`char`**         | 1 byte           | `-128`                                | `127`                                  | Karakter ASCII atau angka kecil yang disimpan sebagai nilai integer bertanda.              |
| **`unsigned char`**| 1 byte           | `0`                                   | `255`                                  | Karakter atau angka non-negatif dalam rentang 0–255.                                        |
| **`wchar_t`**      | 2–4 bytes        | Platform-dependent                    | Platform-dependent                     | Karakter wide untuk teks Unicode.                                                          |
| **`char8_t`**      | 1 byte           | `0`                                   | `255`                                  | Karakter dalam encoding UTF-8 (C++20).                                                     |
| **`char16_t`**     | 2 bytes          | `0`                                   | `65535`                                | Karakter dalam encoding UTF-16 untuk Unicode.                                              |
| **`char32_t`**     | 4 bytes          | `0`                                   | `4294967295`                           | Karakter dalam encoding UTF-32 untuk Unicode.                                              |
| **`short`**        | 2 bytes          | `-32,768`                             | `32,767`                               | Bilangan bulat kecil bertanda.                                                             |
| **`unsigned short`**| 2 bytes         | `0`                                   | `65,535`                               | Bilangan bulat kecil non-negatif.                                                          |
| **`int`**          | 4 bytes          | `-2,147,483,648`                      | `2,147,483,647`                        | Tipe integer default. Digunakan dalam operasi sehari-hari.                                 |
| **`unsigned int`** | 4 bytes          | `0`                                   | `4,294,967,295`                        | Bilangan bulat non-negatif yang lebih besar dari `int`.                                    |
| **`long`**         | 4 bytes          | `-2,147,483,648`                      | `2,147,483,647`                        | Mirip dengan `int` (dalam arsitektur 32-bit).                                              |
| **`unsigned long`**| 4 bytes          | `0`                                   | `4,294,967,295`                        | Bilangan bulat non-negatif lebih besar dari `long`.                                        |
| **`long long`**    | 8 bytes          | `-9,223,372,036,854,775,808`          | `9,223,372,036,854,775,807`            | Bilangan bulat ekstra besar. Cocok untuk aplikasi presisi tinggi.                          |
| **`unsigned long long`** | 8 bytes   | `0`                                   | `18,446,744,073,709,551,615`           | Bilangan ekstra besar non-negatif.                                                        |
| **`float`**        | 4 bytes          | ~`-3.4 × 10^38`                       | ~`3.4 × 10^38`                         | Angka desimal presisi tunggal untuk grafik atau game.                                       |
| **`double`**       | 8 bytes          | ~`-1.7 × 10^308`                      | ~`1.7 × 10^308`                        | Angka desimal presisi ganda untuk aplikasi ilmiah.                                         |
| **`long double`**  | 10–16 bytes      | ~`-1.1 × 10^4932`                     | ~`1.1 × 10^4932`                       | Angka desimal dengan presisi sangat tinggi. Cocok untuk simulasi numerik atau data besar.   |
| **`void`**         | -                | -                                     | -                                      | Tipe khusus tanpa ukuran atau nilai. Untuk fungsi tanpa return value.                     |
| **`nullptr_t`**    | -                | -                                     | -                                      | Pointer null, digunakan untuk nilai pointer kosong.                                        |
| **`int8_t`**       | 1 byte           | `-128`                                | `127`                                  | Bilangan bulat kecil dengan tanda (C++11).                                                 |
| **`uint8_t`**      | 1 byte           | `0`                                   | `255`                                  | Bilangan bulat kecil tanpa tanda (C++11).                                                  |
| **`int16_t`**      | 2 bytes          | `-32,768`                             | `32,767`                               | Bilangan bulat sedang dengan tanda (C++11).                                                |
| **`uint16_t`**     | 2 bytes          | `0`                                   | `65,535`                               | Bilangan bulat sedang tanpa tanda (C++11).                                                 |
| **`int32_t`**      | 4 bytes          | `-2,147,483,648`                      | `2,147,483,647`                        | Bilangan bulat besar dengan tanda (C++11).                                                 |
| **`uint32_t`**     | 4 bytes          | `0`                                   | `4,294,967,295`                        | Bilangan bulat besar tanpa tanda (C++11).                                                  |
| **`int64_t`**      | 8 bytes          | `-9,223,372,036,854,775,808`          | `9,223,372,036,854,775,807`            | Bilangan bulat ekstra besar dengan tanda (C++11).                                          |
| **`uint64_t`**     | 8 bytes          | `0`                                   | `18,446,744,073,709,551,615`           | Bilangan bulat ekstra besar tanpa tanda (C++11).                                           |
| **`size_t`**       | Platform-dependent | `0`                                | Platform-dependent                     | Untuk menyimpan ukuran objek atau array.                                                  |

# Penjelasan Tipe Data di C++ dengan Kebutuhan Khusus

## 1. **Tipe Data Integer**

| **Tipe Data**             | **Kenapa Diperlukan**                                     | **Cara Kerja**                                               | **Perhatian Penggunaan**                                                | **File Header**      | **Kapan Penggunaannya**                                         | **Kelebihan**                                               | **Kekurangan**                                              | **Versi C++ yang Mendukung** | **Kebutuhan Khusus**                                       |
|---------------------------|-----------------------------------------------------------|-------------------------------------------------------------|-----------------------------------------------------------------------|----------------------|----------------------------------------------------------------|-------------------------------------------------------------|------------------------------------------------------------|-----------------------------|------------------------------------------------------------|
| `int`                      | Menyimpan nilai integer (bilangan bulat).                 | Menyimpan bilangan bulat, biasanya 4 byte di banyak platform. | Bisa terjadi overflow jika nilai lebih besar dari kapasitas. | Tidak ada              | Digunakan untuk perhitungan numerik yang tidak melibatkan desimal. | Cepat dan efisien untuk perhitungan bilangan bulat.          | Rentan terhadap overflow jika melebihi batas tipe.         | C++98 dan lebih baru         | -                                                          |
| `unsigned int`             | Menyimpan bilangan bulat tanpa tanda (positif).           | Sama dengan `int` tetapi hanya menyimpan nilai positif.       | Tidak dapat menyimpan nilai negatif.                                 | Tidak ada              | Digunakan untuk nilai yang pasti positif seperti hitungan populasi. | Menyediakan rentang nilai yang lebih luas dibandingkan dengan `int`. | Tidak bisa menangani nilai negatif.                       | C++98 dan lebih baru         | -                                                          |
| `long`                     | Menyimpan bilangan bulat dengan rentang lebih besar dari `int`. | Memiliki ukuran 4 atau 8 byte, bergantung pada platform.      | Bisa terjadi overflow jika nilai lebih besar dari kapasitas. | Tidak ada              | Untuk nilai integer yang lebih besar dari `int`.               | Menyediakan rentang yang lebih besar dibandingkan dengan `int`. | Memiliki overhead lebih besar di beberapa platform.        | C++98 dan lebih baru         | -                                                          |
| `unsigned long`            | Menyimpan bilangan bulat positif dengan rentang lebih besar. | Sama dengan `long` tetapi hanya menyimpan nilai positif.       | Tidak bisa menyimpan nilai negatif.                                 | Tidak ada              | Digunakan ketika nilai selalu positif dan sangat besar.        | Rentang nilai lebih besar dibandingkan dengan `unsigned int`. | Tidak bisa menangani nilai negatif.                       | C++98 dan lebih baru         | -                                                          |
| `long long`                | Menyimpan nilai bilangan bulat yang sangat besar.         | Memiliki ukuran 8 byte pada sebagian besar platform.         | Bisa terjadi overflow jika nilai lebih besar dari kapasitas. | Tidak ada              | Untuk nilai integer yang sangat besar.                         | Rentang sangat besar untuk bilangan bulat.                  | Memiliki overhead lebih besar.                             | C++11 dan lebih baru         | -                                                          |
| `unsigned long long`       | Menyimpan bilangan bulat positif yang sangat besar.       | Sama dengan `long long` tetapi hanya menyimpan nilai positif. | Tidak bisa menyimpan nilai negatif.                                 | Tidak ada              | Untuk nilai sangat besar yang hanya positif.                   | Rentang sangat besar dibandingkan dengan `unsigned int`.     | Tidak bisa menangani nilai negatif.                       | C++11 dan lebih baru         | -                                                          |
| `int8_t`                   | Menyimpan bilangan bulat 8-bit (signed).                  | Menyimpan nilai bilangan bulat 8-bit, dari -128 hingga 127.  | Terbatas pada 8 bit, rentang sempit.                                   | `<cstdint>`           | Untuk data dengan ukuran kecil, efisien dalam penggunaan memori. | Efisien dalam penggunaan memori.                              | Rentang nilai terbatas, overflow lebih mudah terjadi.      | C++11 dan lebih baru         | -                                                          |
| `uint8_t`                  | Menyimpan bilangan bulat 8-bit (unsigned).                | Menyimpan nilai bilangan bulat 8-bit, dari 0 hingga 255.      | Tidak bisa menyimpan nilai negatif.                                   | `<cstdint>`           | Digunakan untuk data kecil yang hanya membutuhkan nilai positif. | Lebih efisien dalam memori daripada tipe data lebih besar.     | Rentang terbatas untuk nilai yang sangat kecil.            | C++11 dan lebih baru         | -                                                          |
| `int16_t`                  | Menyimpan bilangan bulat 16-bit (signed).                 | Menyimpan nilai bilangan bulat 16-bit, dari -32,768 hingga 32,767. | Terbatas pada 16 bit, overflow mudah terjadi.                        | `<cstdint>`           | Digunakan ketika hanya membutuhkan rentang bilangan kecil.     | Efisien untuk data kecil.                                    | Rentang terbatas, overflow mungkin terjadi.                | C++11 dan lebih baru         | -                                                          |
| `uint16_t`                 | Menyimpan bilangan bulat 16-bit (unsigned).               | Menyimpan nilai bilangan bulat 16-bit, dari 0 hingga 65,535.   | Tidak bisa menyimpan nilai negatif.                                   | `<cstdint>`           | Digunakan untuk data yang selalu positif dan kecil.            | Lebih efisien dalam memori.                                  | Tidak bisa menangani nilai negatif.                        | C++11 dan lebih baru         | -                                                          |
| `int32_t`                  | Menyimpan bilangan bulat 32-bit (signed).                 | Menyimpan nilai bilangan bulat 32-bit, dari -2,147,483,648 hingga 2,147,483,647. | Terbatas pada 32 bit, overflow bisa terjadi.                          | `<cstdint>`           | Digunakan ketika data 32-bit diperlukan.                       | Memiliki rentang yang cukup besar.                           | Rentang terbatas, overflow mungkin terjadi.                | C++11 dan lebih baru         | -                                                          |
| `uint32_t`                 | Menyimpan bilangan bulat 32-bit (unsigned).               | Menyimpan nilai bilangan bulat 32-bit, dari 0 hingga 4,294,967,295. | Tidak bisa menyimpan nilai negatif.                                   | `<cstdint>`           | Untuk data yang selalu positif dan membutuhkan 32-bit.          | Lebih efisien dalam memori dibandingkan `long`.              | Tidak bisa menyimpan nilai negatif.                        | C++11 dan lebih baru         | -                                                          |
| `int64_t`                  | Menyimpan bilangan bulat 64-bit (signed).                 | Menyimpan nilai bilangan bulat 64-bit, dari -9,223,372,036,854,775,808 hingga 9,223,372,036,854,775,807. | Terbatas pada 64 bit, overflow bisa terjadi.                          | `<cstdint>`           | Digunakan untuk data 64-bit yang besar.                        | Rentang yang sangat besar.                                    | Menggunakan lebih banyak memori.                          | C++11 dan lebih baru         | -                                                          |
| `uint64_t`                 | Menyimpan bilangan bulat 64-bit (unsigned).               | Menyimpan nilai bilangan bulat 64-bit, dari 0 hingga 18,446,744,073,709,551,615. | Tidak bisa menyimpan nilai negatif.                                   | `<cstdint>`           | Digunakan untuk data 64-bit yang hanya positif.                | Rentang sangat besar.                                         | Tidak bisa menangani nilai negatif.                        | C++11 dan lebih baru         | -                                                          |
